# mysql-sales-analysis
Title - Business Insights from Pizza Sales Dataset Using MySQL .

Objective - To analyze pizza sales transactions using SQL in order to uncover 
insights on sales trends, customer preferences, revenue contribution, and 
business performance. 

Dataset - File Format: CSV

Columns: order_details_id , order_id , pizza_id , quantity , date , time , pizza_type , name , category , ingredients , size , price .

Description - Dataset contains pizza sales transactions with details on orders, pizzas, categories, sizes, ingredients, prices, and timestamps.

Process - Performed SQL-based analysis using Joins, Aggregations, Group By, and Date-Time functions.Executed queries to calculate total orders, revenue, top-selling pizzas, popular sizes, and category-wise sales distribution.
Analyzed sales performance across time, product category, and pizza types to identify customer preferences and revenue drivers.

Analysis Covered 
Basic

Retrieve the total number of orders placed.

Calculate the total revenue generated from pizza sales.

Identify the highest-priced pizza.

Identify the most common pizza size ordered.

List the top 5 most ordered pizza types along with their quantities.

Intermediate

Find the total quantity of each pizza category ordered.

Determine the distribution of orders by hour of the day.

Category-wise distribution of pizzas.

Group the orders by date and calculate the average number of pizzas ordered per day.

Determine the top 3 most ordered pizza types based on revenue.

Advanced

Calculate the percentage contribution of each pizza type to total revenue.

Analyze the cumulative revenue generated over time.

Determine the top 3 most ordered pizza types based on revenue for each category.

SQL Concepts & Functions - Joins (INNER JOIN) , Aggregations (SUM, COUNT, AVG) , GROUP BY & ORDER BY , Date & Time Functions (HOUR, DATE) ,LIMIT & Aliasing , Nested Queries / Subqueries , Revenue & Percentage Calculations

How to Run - Clone this repository
Open MySQL Workbench (or any SQL client)
Create a database and import the provided CSV dataset 
Open the SQL script file:
Pizza Sales.sql
Run the queries step by step to generate insights .
